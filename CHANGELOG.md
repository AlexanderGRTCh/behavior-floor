# Changelog

All notable changes to this project will be documented in this file.

The format follows Keep a Changelog.

## [0.1.0] 2026-07-22

### Added

- Initial release of Behavior Floor.
- Rules floor, communication profiles, delegation doctrine, and editable persona templates.
- `SessionStart` and `UserPromptSubmit` hooks with a `python`, `python3`, `py` fallback chain.
- Three slash commands: `/behavior-floor:rules`, `/behavior-floor:comms`, and `/behavior-floor:floor`.
- Verification on Windows, macOS, and Linux.
